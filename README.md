# Coding challenges
A collection of coding challenges to practice datastructures, algorithms and syntax from different programming languages.

It is divided between logic problems, data structures and algorithms, which can be accessed through each different folder. 

## 📂 Repository structure
- `logic`: contains solved logic problems from [DSA Tutorial](https://www.geeksforgeeks.org/dsa/logic-building-problems/)
- `problems`: contains solved problems from coding challenges from different sources
